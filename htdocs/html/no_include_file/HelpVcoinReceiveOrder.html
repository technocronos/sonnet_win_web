出金依頼するときは「受取先ウォレットアドレス」に自分のウォレットのアドレスを入力するのだ。
その際、手入力は厳禁なのだ。必ずコピペするのだ。

アドレスが間違っていた場合、出金できないのだ。間違ってたアドレスに送金した場合、金が戻って来ない場合があるのだ。その場合は再送はできないのだ

出金は今ある残高全部になるのだ。いくら出金するかは選択はできないのだ。

送金手数料は0.0005BTCかかるのだ。これはユーザー負担になるから実際受け取れる額は「残高-送金手数料」になるのだ。
仮想通貨を送るには必ず手数料がかかるのは仕方ないのだ。

不正防止のために運営は送金するときには必ずプレイログを確認するのだ。整合性が取れない場合は出金はできないのだ。

ビットコインは運営があらかじめ用意しておいたものを送るのだ。ま、持ち出しなのだ・・。いくら用意してるかはあらかじめ告知してこれが尽きたらキャンペーンは終了になるのだ。

キャンペーン終了、というのは「モンスター討伐なりバトルイベントなりでビットコインが排出されなくなる」という意味であって溜まったビットコインの出金依頼が出来なくなる、という意味じゃないからそこは安心するのだ。

手数料がもったいないなら、キャンペーン終了まで溜めて終了したらゆっくり出金依頼かければいいのだ。
