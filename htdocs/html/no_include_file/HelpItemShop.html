アイテムや装備はショップで{$smarty.const.GOLD_NAME}を使って買うのだ｡場所や条件が変わると売ってるものが変わるのだ｡

{$smarty.const.PLATFORM_CURRENCY_NAME}で買い物するなら､いつでもどこでも同じもの買えるのだ｡{$smarty.const.PLATFORM_CURRENCY_NAME}でしか買えない物もたくさんあるのだ｡

ガチャショップっていうのもあるのだ｡例によって{$smarty.const.PLATFORM_CURRENCY_NAME}なのだけど､フリーチケットあるならタダでできるのだ｡やっぱり例によって､ガチャにはいいものがあるのだ｡
