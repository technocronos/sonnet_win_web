ビットコインを受け取るにはウォレットアドレスってのが必要なのだ
これはユーザーが自分で用意しなければいけないのだ

ビットコインを取り扱っている取引所で口座を開くのもいいし、とりあえず受け取りたいなら手軽にビットコインのウォレットだけ作っちゃってもいいのだ。
{if $smarty.const.PLATFORM_TYPE == "geso"}ここはゲソてんということもあるし、GMOコインとかオススメしとくのだ
{/if}
国内の取引所だと身分証明書とかいろいろ大変というユーザーは海外の取引所、例えばblockchainとかならとりあえずウォレット作るくらいのことはできるのだ。でも日本円に換金したきゃやっぱ国内にしとくのだ

その際はビットコインを受け取れるウォレットじゃないとビットコインは受け取れないのだ。ほかにいろいろ仮想通貨があるけどほかの仮想通貨のアドレス指定しても送れないのだ。

また、ビットコインを受け取るには最低でも{$smarty.const.VCOIN_MINIMAM}BTC以上のビットコインをゲットしてないと受け取ることはできないのだ

また、ビットコインを受け取るには最低でも{$smarty.const.VCOIN_MINIMAM_PAYMENT}円以上の課金してないと受け取ることはできないのだ
これは景表法という法律上、一定以上物品を購入していない人に金銭に類するものをタダで渡すことはできないからなのだ。イジワルしてるわけじゃないのだ。

これらの条件を満たしたら、自分のウォレットに出金依頼することができるのだ

もう一回まとめると
１，ビットコインアドレスを持っている
２，0.02BTC以上持っている
３，これまで5000円以上課金している
 なのだ。
