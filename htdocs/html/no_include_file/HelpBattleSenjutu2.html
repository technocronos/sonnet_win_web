まずは相手によりなのだ｡
スピードで大きく勝ってるんなら吸収成功しやすいのだから､吸収狙うのがいいと思うのだ｡
逆にスピードが大きく負けてると､吸収選択してもあんま期待できないのだ｡

そんで､状況によりなのだ｡

攻撃カードの属性が同じなら､同属性でも無効にできる強攻がセオリーなのだ｡
でもそう考えるのは相手も同じなのだ｡相手も強攻してくるのだから､スピード勝ってるなら裏をかいて吸収するのもアリなのだ｡
慎重は…裏の吸収の､さらに裏をかきたいときくらいなのだ?

属性勝ちしてるなら…まず吸収はあんま意味ないのだ｡属性で相手の攻撃無効にしちゃうのに､吸収狙っててもしょーがないのだ｡
ま､慎重で着実にダメージを与えるのがセオリーなのだ?
強攻でよりダメージを増やしてもいいのだけど､相手がスピード速いときに吸収選択されるとモッタイナイ結果になるのだ｡

属性負けしてると悩ましいのだ…吸収で被害減少を狙うか､属性負けでも耐えられる場合がある強攻で少しでも反撃を狙うか…なのだ｡
慎重は…なんだか強攻も吸収も期待できないネガティブな気分のときに選ぶといいのだ｡ちょっぴりだけど確実にダメージ軽くなるのだ｡
